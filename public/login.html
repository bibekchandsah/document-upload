How to Create a GitHub Token

  1. Go to GitHub Settings

    Click on your profile picture in the top right corner, then select Settings

  2. Open Developer Settings

    Scroll down and click on Developer Settings in the left sidebar

  3. Select Personal Access Tokens

    Click on Personal access tokens in the left menu

  4. Choose Tokens (classic)

    Select Tokens (classic) from the submenu

  5. Generate New Token

    Click the Generate new token button, then select Generate new token (classic)

  6. Add Token Name

    Write a descriptive name for your token in the Note field (e.g., "Document Viewer")

  7. Set Expiration Date

    Choose an Expiration Date according to your preference. For security, 30 days is recommended

  8. Select Scopes

    Under Select scopes, check the repo checkbox for read & write access to repositories

  9. Generate Token

    Scroll to the bottom and click Generate token

  10. Copy and Save

    Important: Copy the token immediately and save it securely. You won't be able to see it again!

Tip: The "repo" scope allows this app to read and write to your repositories. Never share your token with anyone!

Direct Link: https://github.com/settings/tokens
